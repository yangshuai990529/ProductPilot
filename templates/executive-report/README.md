# 模板 - 管理层汇报演示模板 (Executive Report Template)

## 1. 使用场景
本模板用于面向公司高管（C-Level）、VP、产品总经理的双周会、季度业务规划汇报，以及年度战略大纲宣讲。

## 2. 页面与章节结构

### 2.1 必选页面/章节
1. **核心结论与商业成果摘要 (Executive Summary)**:
   - 绝不拖泥带水，开门见山汇报当前的核心成果与需要高管决策的关键点。
2. **大势背景与核心问题 (Context)**:
   - 用 1–2 页极简且有数据支撑的幻灯片，讲清面临的痛点。
3. **高阶解决方案与用户价值 (High-level Solution)**:
   - 核心方案概念设计图，避开技术细节，聚焦用户收益与业务增长。
4. **关键里程碑路线图 (Milestones)**:
   - 粗颗粒度的业务上线计划。
5. **需要的支持与决策事项 (Asks & Decision)**:
   - 需要管理层批复的预算、人力支持或需要拍板的路线选择。

## 3. 推荐表达图表
- 使用 **大号字体的核心指标卡片**（大图大字）。
- 使用 **简明横向里程碑轴线**。
- 使用 **两难决策对比表**（方案 A 优缺点 vs 方案 B 优缺点）。

## 4. 输出检查列表 (Checklist)
* [ ] 演示文稿是否做到“一页仅表达一个核心结论”？
* [ ] 是否彻底去除了所有大段的文字段落与底层技术术语？
* [ ] 是否在最后明确列举了需要高管拍板的决策项？
