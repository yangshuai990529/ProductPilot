# 模板 - 用户与行业研究报告模板 (Research Report Template)

## 1. 使用场景
本模板适用于新市场探索阶段的行业趋势洞察、细分目标用户痛点调研以及产品立项前的可行性研究。

## 2. 页面与章节结构

### 2.1 必选页面/章节
1. **研究背景与核心课题 (Objectives)**:
   - 为什么要进行此项研究？要解决什么商业疑问？
2. **行业趋势与市场数据 (Market Insights)**:
   - 行业整体出货量、复合增长率、主流品牌市占率（注：必须标明权威第三方数据出处）。
3. **典型用户画像 (User Personas)**:
   - 目标客群细分、使用习惯描述、最迫切需要被解决的痛点场景。
4. **用户体验旅程图 (User Journey Map)**:
   - 还原用户全流程操作，提炼各节点的用户情绪变化与痛点。
5. **产品转化机会点 (Product Application)**:
   - 研究成果如何指导接下来的产品方案设计。

## 3. 推荐表达图表
- 使用 **用户旅程矩阵** 展现痛点分布。
- 使用 **折线图/柱状图** 表达市场规模变化。
- 使用 **价值象限** 标明产品机会的紧急度与重要度。

## 4. 输出检查列表 (Checklist)
* [ ] 是否包含了至少 2 个细分客群画像？
* [ ] 报告中的核心数据是否均注明了出处（如 Canalys 2026 Q1）？
* [ ] 研究结论是否具备可被产品落地执行的承接性？
