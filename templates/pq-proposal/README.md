# 模板 - 画质产品方案模板 (PQ Product Proposal Template)

## 1. 使用场景
本模板适用于电视、投影仪及显示设备的高级画质（PQ）引擎开发、画质算法集成（如局部控光、运动补偿）及画质设置菜单迭代。

## 2. 页面与章节结构

### 2.1 必选页面/章节
1. **面板规格与主芯片约束 (Hardware Prerequisites)**:
   - 物理背光分区数、峰值亮度、色域指标、SoC 硬件解码规格。
2. **画质信号路由策略 (Signal Routing)**:
   - 不同输入格式（SDR / HDR10 / Dolby Vision）与不同通道下，画质引擎的跳转与默认生效的图像模式。
3. **用户画质设置菜单树 (PQ Settings Menu)**:
   - 详细列出设置项（亮度、色温、降噪等）的调节范围、步长及互斥逻辑。
4. **性能指标与测试验证 (KPI & Validation)**:
   - 算法开启后的系统延迟要求、调校 Delta E 标准。

### 2.2 可选页面/章节
- 杜比视界/HDR10+ 官方认证（Dolby Certification）前置依赖审计表。

## 3. 推荐表达图表
- 使用 **参数路由拓扑图** 展现信号处理流水线（Pipeline）。
- 使用 **菜单树结构图** 描述图像设置层级。
- 使用 **对比矩阵** 展现不同画质模式下的参数差异。

## 4. 输出检查列表 (Checklist)
* [ ] 画质设置项的默认值与步长是否具备完整定义？
* [ ] 画质算法与低延迟“游戏模式”的互斥规则是否清晰？
* [ ] 是否明确定义了音视频延迟对齐标准？
