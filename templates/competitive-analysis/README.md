# 模板 - 竞品分析模板 (Competitive Analysis Template)

## 1. 使用场景
本模板用于对标直接竞争对手的功能特性，进行 SWOT 态势分析，提炼我方防守点与差异化进攻点，辅助产品战略定位。

## 2. 页面与章节结构

### 2.1 必选页面/章节
1. **对比竞品画像 (Competitor Profile)**:
   - 选取哪些对标竞品？竞品的核心商业模式和技术标签是什么？
2. **多维度对比矩阵 (Feature Comparison Matrix)**:
   - 从功能覆盖率、交互易用性、性能指标（如内存、时延）进行横向表格对比。
3. **SWOT 分析 (SWOT Analysis)**:
   - 梳理我方所面临的优势（S）、劣势（W）、机会（O）、威胁（T）。
4. **防守与进攻战略 (Strategy)**:
   - 明确哪些是“必须追平（To-Be-Equal）”的基础底座功能，哪些是“形成优势（To-Be-Better）”的拳头功能。

## 3. 推荐表达图表
- 使用 **横向对比表格** (带勾叉或星级标识) 展现矩阵。
- 使用 **2x2 SWOT 矩阵卡片**。
- 使用 **雷达图** 展现我方与竞品多维能力的强弱对比。

## 4. 输出检查列表 (Checklist)
* [ ] 是否明确避开了主观偏见，采用客观数据或可还原的实测结论？
* [ ] 对竞品的缺点描述是否有事实依据？
* [ ] 对我方劣势的评估是否足够坦诚且给出了弥补方案？
