# ProductPilot 产品模板系统

本目录包含产品经理在日常工作流中高频使用的各类文档与汇报模板。所有的模板在生成具体内容时，都必须注入规范的文字层级（L1–L6），并遵循本仓库的设计规范。

## 模板目录结构

```
templates/
├── README.md                 # 本文档
├── product-proposal/         # 通用产品立项方案模板
├── ai-proposal/              # AI/Agent 产品方案模板
├── pq-proposal/              # 画质 (Picture Quality) 产品方案模板
├── dfs-proposal/             # DFS/无线射频规格方案模板
├── research-report/          # 用户与行业研究报告模板
├── competitive-analysis/     # 竞品分析矩阵模板
├── technical-proposal/       # 技术对接与系统需求 (SR) 模板
├── executive-report/         # 管理层汇报演示文稿模板
├── roadmap/                  # 产品中长期路线图模板
└── weekly-report/            # 周报与标准作业程序 (SOP) 模板
```

## 使用规则

每个模板目录中均有说明文件，定义了：
1. **使用场景 (Scenarios)**：该模板在何种评审/汇报级别下使用。
2. **页面与章节结构 (Structure)**：哪些是必选（Mandatory）章节，哪些是可选（Optional）章节。
3. **推荐图表 (Recommended Visuals)**：推荐配以流程图、对比矩阵还是时序图。
4. **输出检查列表 (Checklist)**：提交评审前必须自检的条目。
