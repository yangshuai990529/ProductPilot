  • 标准 / 影院 / 体育 / 游戏 / 图形 / 生动 / 自定义
  • Netflix 校准模式（Netflix Calibrated Mode）
  • 影院专业 / 影院家庭（双影院模式区分）
    • 核心功能项
  • 亮度 / 色彩 / 色调 / 锐度
  • Reality Creation（真实创建）
  • 平滑渐变 / 随机噪点降噪 / 数字降噪
  • Motionflow / CineMotion（运动补偿与电影还原）
  • HDR 模式 / Live Color / 色温

  • 缺少独立游戏面板，游戏参数分散在多个层级

    LG 画质菜单概览

    竞品分析 · LG（webOS）
    机密材料，请勿外传
    LG
    LG — webOS 系统
    OLED 画质调校 + Game Optimizer 游戏中心
    系统特征
    • 基于 webOS 的独立自研系统，菜单逻辑区别于 Google TV
    • 画质设置集中于「全部设置 → 画面」与「游戏优化器」两个入口
    • OLED 原生优势：像素级控光，无需独立局域控光分区设置
    • AI Picture Pro 由 α9 / α11 处理器驱动，支持场景识别
    核心画质模块
    AI 画质 Pro OLED 像素控光 Filmmaker Mode 游戏优化器
    LG 画质菜单特点
    • 双入口设计：通用画质走「画面」菜单，游戏画质独立走「游戏优化器」
    • OLED 专属项：包含「OLED 像素亮度」「OLED 运动平滑度」等 OLED 特有参数
    • 杜比视界 IQ：内置杜比视界与杜比视界 IQ（随环境光自适应）待验证
    • 白平衡 2 点 + 20 点：提供 ISF 专业模式下 20 点白平衡校准待验证
    • Calman 联合校准：支持 Calman 自动校准（AutoCal）待验证
    相对短板
    • CMS 色彩管理隐藏较深，普通用户不易触达
    • webOS 菜单层级相对分明，但专业项分类不够直观
    • 国内本地化资源（相比海信）相对薄弱
