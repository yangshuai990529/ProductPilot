# ProductPilot 输出产物目录 (Output)

本目录为 ProductPilot 运行过程中自动生成的 PPT 幻灯片结构、PRD/SR 文档和原型的输出目标区。

## 目录结构

```
output/
├── README.md               # 本文档
├── ppt/                    # 输出的 PPT 汇报材料
├── docs/                   # 输出的 PRD 与 SR 产品文档
├── prototypes/             # 输出的交互式 HTML 原型
└── images/                 # 输出的 AI 生成图片资源
```

> [!IMPORTANT]
> 默认情况下，本目录下的子内容（`.gitkeep` 除外）均已被 `.gitignore` 规则拦截，确保在本地测试生成的各种项目文件不会意外上传到开源代码库中。
