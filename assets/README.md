# ProductPilot 静态资源目录 (Assets)

本目录用于存放在编写原型、文档与演示幻灯片时引用的本地静态图像与矢量图标。

## 目录结构

```
assets/
├── README.md               # 本文档
├── icons/                  # 常用 SVG 矢量图标
├── illustrations/          # AI 生成的高品质插画
├── screenshots/            # 产品真实截图
└── diagrams/               # 导出的流程图与架构图
```

> [!NOTE]
> 在 Markdown 中引用本目录下的资源时，必须使用**相对路径**（例如 `../../assets/icons/home.svg`），禁止使用绝对路径。
