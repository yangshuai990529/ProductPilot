# 知识库 - TV 大屏端十英尺界面交互指南

* **版本状态**: Active
* **适用范围**: 所有大屏电视、投影仪及机顶盒应用
* **数据来源**: 行业通用十英尺设计指南
* **更新时间**: 2026-07-11

## 核心定义
**十英尺交互（10-foot UI）** 是指用户在距离屏幕约 3 米（10 英尺）远的地方操作电视设备时的交互界面。与手机和电脑不同，用户不能进行手势触摸或精确鼠标点击，必须依赖遥控器的十字方向键（D-Pad）完成导航。

## 交互设计原则

1. **焦点唯一性与明确性**：
   - 屏幕上任何时候有且仅有一个处于“聚焦（Focused）”状态的元素。
   - 焦点必须在视觉上有非常清晰的反馈，例如：等比例放大 1.05–1.1 倍、发光描边、阴影悬浮。
2. **遥控器十字键寻路（D-Pad Navigation）**：
   - 焦点只能沿上、下、左、右四个维度移动。
   - 避免对角线排列的卡片或重叠组件，这会导致遥控器寻路路径不明确（非直角位移）。
   - 禁止“焦点死胡同”，即用户在某个界面卡片上按下任意方向键都无法离开当前位置。
3. **极简操作链路**：
   - 电视端用户耐心较低。从开机到核心内容播放的点击次数（Clicks）应当控制在 3 次以内。
   - 关键操作（如：切换信号源、搜索）需提供快捷物理按键映射。
4. **字体大小与可见度基线**：
   - 大屏下视距较远。在 1080p（1920×1080）画布下，正文字号不得小于 24 px，辅助字号不得小于 20 px。低于 18 px 的文字在大屏上将完全无法阅读。
