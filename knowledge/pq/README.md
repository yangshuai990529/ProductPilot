# 知识库 - 画质 (Picture Quality) 技术标准

* **版本状态**: Active
* **适用范围**: 支持 HDR 与 4K 高级显示芯片的画质电视产品
* **数据来源**: UHD 联盟规格书、SoC 画质寄存器基线
* **更新时间**: 2026-07-11

## 核心定义
**画质（Picture Quality，PQ）** 是指图像在显示设备上呈现给用户的综合视觉还原度。它由分辨率、色彩深度、对比度、亮度范围和动态补偿算法等多个维度共同决定。

## 关键画质规格标准

### 1. 高动态范围 (High Dynamic Range，HDR)
* **HDR10**: 行业最通用的开放式 HDR 格式，采用静态元数据（Static Metadata）。最大峰值亮度定义为 1000–4000 nits，色域基准为 BT.2020。
* **HDR10+**: 由三星等企业主导的动态元数据（Dynamic Metadata）格式，可逐帧（Frame-by-Frame）优化场景亮度和对比度。
* **HLG (Hybrid Log-Gamma)**: 广电直播常用的 HDR 格式，兼容 SDR（标准动态范围）电视，无需元数据传输。

### 2. 色彩与色深 (Color Depth)
* **10-bit 色深**: 能呈现 10.7 亿种色彩，在大面积渐变天空或暗光场景下，可彻底避免 8-bit 的色彩断层（Banding）现象。
* **BT.2020**: 超高清电视的色彩标准空间，色域范围远大于普通的 Rec.709 空间。画质产品指标通常以 DCI-P3 覆盖率（如 95% DCI-P3）为准。

### 3. 动态画质算法
* **运动估计与运动补偿 (MEMC)**: 通过插帧算法，将 24 fps 或 30 fps 的低帧率源提升至 60 fps 或 120 fps，有效消除大屏快速运动场景下的抖动（Judder）与模糊（Blur）。
* **局部背光控制 (Local Dimming)**: 直下式 MiniLED 背光电视的核心算法。通过将背光划分为数百个独立控制的分区，配合画面内容进行亮暗调节，实现极高对比度（百万比一）并大幅压低暗场漏光。
