# 知识库 - DFS (动态频率选择) 与 Wi-Fi 频段规范

* **版本状态**: Active
* **适用范围**: 所有集成 Wi-Fi 5/6/7 的智能终端设备
* **数据来源**: 工业和信息化部无线电管理局 (SRRC)、美国联邦通信委员会 (FCC)
* **更新时间**: 2026-07-11

## 核心定义
**动态频率选择（Dynamic Frequency Selection，DFS）** 是一种无线局域网（WLAN）的避让共享技术。在 5GHz 频段中，某些无线信道（如 52–64 信道）与民用气象雷达或军事雷达的工作频段重合。法律强制规定：工作在这些信道上的 Wi-Fi 设备如果检测到雷达信号，必须在极短时间内（通常是 10 秒内）自动切离当前信道，并在此后一段时间内禁止在此信道上传输。

## 核心技术要求

### 1. 信道可用性检查 (CAC)
* 在设备开始在 DFS 信道上传输前，必须对当前信道进行雷达信号监听。
* 监听时间（CAC Time）通常为 **60 秒**。在此期间设备只能收听，禁止发送任何 Wi-Fi 帧。
* 若检测到雷达，该信道自动置为不可用，设备必须重新选频。

### 2. 避让时间 (Move & Closing Time)
* 一旦运行中检测到雷达：
  - **信道移动时间 (Channel Move Time)**: 必须在 **10 秒** 内停止在当前信道的一切发射。
  - **关机传输时间 (Channel Closing Transmission Time)**: 10 秒过渡期内，控制/管理帧的总传输时长不能超过 **200 毫秒**，以防雷达接收机被干扰致盲。

### 3. 非占用期 (NOP)
* 触发避让的信道进入 **30 分钟** 的锁定状态（Non-Occupancy Period，NOP）。在此期间，无论如何重启或重置，设备都禁止重新尝试将此信道用作主/副载波。

## 国家射频准入差异 (5GHz 频段)
* **中国 (SRRC)**: 5GHz 高频段（149–165）为非 DFS 频段。雷达避让信道主要集中在 5GHz 低频段中的 52–64 信道。
* **美国 (FCC)**: 52–64 以及 100–144 均为 DFS 频段，测试要求极为严苛。
