# 知识库 - 产品术语表 (Product Glossary)

* **版本状态**: Active
* **适用范围**: 整个 ProductPilot 仓库与生成的交付物
* **更新时间**: 2026-07-11

本术语表用于规范项目内使用的专业中文名称与其英文对照，所有输出应保持用词一致性。

## 核心术语列表

| 中文名称 | 英文名称与缩写 | 定义与应用场景 |
| :--- | :--- | :--- |
| 产品需求文档 | Product Requirements Document (PRD) | 面向业务、交互与功能的标准产品开发指南。 |
| 系统需求规格书 | System Requirement (SR) | 将 PRD 技术化的系统间及接口交互底层契约。 |
| 十英尺交互界面 | 10-foot User Interface (10-foot UI) | 用户距离大屏电视 3 米远时所使用的电视界面。 |
| 动态频率选择 | Dynamic Frequency Selection (DFS) | Wi-Fi 5GHz 频段雷达避让的法定机制。 |
| 运动估计与运动补偿 | Motion Estimation, Motion Compensation (MEMC) | 用于提升电视画面流畅度的插帧画质算法。 |
| 检索增强生成 | Retrieval-Augmented Generation (RAG) | 大模型配合外部向量数据库知识的检索机制。 |
| 可变刷新率 | Variable Refresh Rate (VRR) | 显示器自适应游戏输出帧率以防止撕裂的技术。 |
| 屏幕调校/校准 | Calibration | 使用专业仪器校正屏幕 Delta E 色差的过程。 |
| 局域控光/背光分区 | Local Dimming | 直下式 LED 动态亮暗控制以提升对比度的算法。 |
| 最小可行产品 | Minimum Viable Product (MVP) | 用最快速度向市场投放核心功能的验证版本。 |
| 验收标准 | Acceptance Criteria (AC) | 判断一个 PRD 功能点是否合格并可上线的条件。 |
