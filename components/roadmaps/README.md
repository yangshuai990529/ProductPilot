# 组件 - 产品路线图 (Roadmaps)

本组件用于规划产品中长期的演进阶段，展示范围（Scope）与愿景规划。

## 1. 三阶段产品路线图模板

```
+---------------------------------------------------------+
|                  ProductPilot 中长期演进路线              |
+---------------------------------------------------------+
|  近期 (MVP / Alpha) |  中期 (V1.0 / Beta) |  远期 (V2.0) |
+---------------------+---------------------+-------------+
|                     |                     |             |
| 1. 核心架构与内置   | 1. 支持外部 RAG 知识| 1. 深度整合 |
|    11个 Skill 建设。|    库检索与向量搜索 |    Jira/    |
|                     |    能力。           |    Confluence|
| 2. 编写本地仓库自检 |                     |    生态。   |
|    敏感信息脚本。   | 2. 支持 PPT 自动导出|             |
|                     |    为标准 PPTX。    | 2. 发布 SaaS|
|                     |                     |    云平台。 |
+---------------------+---------------------+-------------+
```

## 2. 设计规范
* 路线图需要以简明的分栏表格或文本卡片形式输出。
* 每个交付要点控制在 12–25 字以内，拒绝大段说明。
