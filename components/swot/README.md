# 组件 - SWOT 分析卡片 (SWOT)

本组件用于进行快速的战略态势评估。

## 1. SWOT 2x2 卡片模板

```
+-----------------------------------+-----------------------------------+
|          优势 (Strengths)         |          劣势 (Weaknesses)        |
+-----------------------------------+-----------------------------------+
| 1. 内置 11 个垂类专家 Skill。       | 1. 暂无云端 RAG 同步数据库。       |
| 2. 直接输出汇报级演示文稿与原型。 | 2. 界面目前仅支持 Markdown 交互。 |
+-----------------------------------+-----------------------------------+
|          机会 (Opportunities)     |          威胁 (Threats)           |
+-----------------------------------+-----------------------------------+
| 1. 开源社区目前缺乏针对 PM 岗的   | 1. 头部 IDE (如 Cursor) 可能会直接|
|    系统级工作流 AI 助手。         |    集成产品需求分析插件。         |
+-----------------------------------+-----------------------------------+
```

## 2. 设计规范
* 在 PPT 中，SWOT 应当以 4 宫格彩色卡片呈现。
* 每一个格子内的条目控制在 2 条以内，语言保持高度精炼。
