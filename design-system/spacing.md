# 设计系统 - 间距与对齐规范 (Spacing)

* **更新时间**: 2026-07-11

一致的间距和精准的对齐是构成高级视觉美感（Premium Aesthetics）的基石。在设计系统和输出原型时，ProductPilot 严格遵循 **8 像素网格系统 (8px Grid System)**。

## 1. 间距比例尺 (Spacing Scale)

所有元素之间的 Margin, Padding 和 Gap 均必须为 `8 px` 的倍数：

| 变量名 | 像素值 (px) | 应用场景 |
| :--- | :--- | :--- |
| `spacing-xs` | 8 | 卡片内部标签与标题的微距、图标与文字的 Gap |
| `spacing-sm` | 16 | 卡片内部内容 Padding、小模块内的子条目间距 |
| `spacing-md` | 24 | 卡片与卡片之间的网格 Gutter、L3 模块标题与内容的间距 |
| `spacing-lg` | 40 | 大模块（L2 与 L3 之间）的垂直 Margin |
| `spacing-xl` | 80 | 大屏安全区页边距（Margins） |

---

## 2. 页面元素对齐规则 (Alignment)
* **左对齐优先**：中文字符阅读习惯为自左向右。页面标题、正文列表等，除封面特殊设计外，统一**左对齐**，不使用分散对齐（防止汉字与英文混排时产生巨大的字间距空隙）。
* **垂直居中**：在卡片、表格行以及 D-Pad 导航栏的容器内，小图标与文字必须在垂直方向上精确居中对齐（Alignment: Center）。
* **基线一致**：多列卡片并排时，顶部卡片标题必须位于同一水平基线上，不允许因为卡片内容多少不同而导致卡片高度参差不齐。
