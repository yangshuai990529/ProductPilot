# 设计系统 - 图片使用与 AI 生成规范 (Images)

* **更新时间**: 2026-07-11

在汇报级 PPT 方案中，“图大于字”是不可动摇的设计原则。本规范定义了图片的使用场景、当缺少真实截图时如何利用 AI 生成高品质占位图的 Prompts，以及绝对禁止的行为。

## 1. 真实图片与占位图分界

* **真实产品截图**:
  - 只有在产品功能已上线或已具备真实高保真 UI 原型时，才可使用真实截图。
* **AI 占位图片**:
  - 在新特性的概念推演阶段，若缺少截图，应在幻灯片中插入 AI 生成图，并加注“**示意图片，后续替换**”的辅助说明（L5 级别字号）。
  - **红线约束**：禁止使用生成的虚假界面冒充真实截图欺骗高管或客户，禁止伪造竞争对手的后台管理页面。

---

## 2. AI 图片生成提示词规范 (Prompts)
为保证图片具备极简、高级的画风，生成的 Prompt 必须遵循以下结构：

### Prompt 结构模板
`[主体物 / 动作] + [使用场景 / 空间环境] + [构图与光影] + [画面视觉风格] + [技术参数与渲染比例]`

### 推荐风格参数
* **风格控制**: "Minimalist, premium 3D rendering, clay material, clean lighting, high-tech aesthetics, soft shadows, studio background." (极简、高级 3D 渲染、粘土质感、干净光影、科技美学、软阴影、影棚背景)。
* **背景色系**: 统一使用与设计系统配色一致的暗灰蓝背景（Dark Slate Blue, HSL 220, 25%, 10%），保证图片插入 PPT 后与整体背景完美融合。
* **比例设置**: `--ar 16:9`。
* **禁忌词**: 避免包含任何英文字母或乱码文字。在 Prompt 中明确指明 `"no text, no letters, no watermark"`。
