# 贡献者行为准则 (Contributor Covenant Code of Conduct)

## 我们的承诺

为了营造一个开放和友好的环境，我们作为贡献者和维护者承诺，不论任何年龄、身体大小、残疾、种族、性别特征、性别认同和表达、经验水平、教育程度、社会经济地位、国籍、个人形象、种族、宗教或性取向，人人参与我们的社区都是一种无骚扰的体验。

我们承诺以有助于建立一个健康、包容和互助的社区的方式进行合作和互动。

## 我们的标准

有助于为我们的社区创造积极环境的行为示例包括：

* 使用欢迎和包容的语言
* 尊重不同的观点和经验
* 优雅地接受建设性的批评
* 专注于对社区最有利的事情
* 对其他社区成员表现出同理心

参与者不能接受的行为示例包括：

* 性语言或图像，以及任何形式的性挑衅或骚扰
* 带有挑衅性的评论（Trolling）、侮辱性/贬损性的评论以及人身或政治攻击
* 公开或私下的骚扰
* 未经明确许可发布他人的私密信息，例如物理地址或电子邮箱地址
* 其他可被合理认为在专业环境中不合适的行为

## 违规行为处理

社区维护者负责澄清和执行我们的可接受行为标准，并将对他们认为不合适、威胁、冒犯或有害的任何行为采取适当和公正的纠正措施。

如果您遇到骚扰或任何其他违反本准则的行为，请联系项目维护团队：`conduct@productpilot.org`。所有投诉都将得到及时调查并妥善处理。
