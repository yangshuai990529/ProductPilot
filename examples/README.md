# ProductPilot 历史案例与示例库 (Examples)

本目录存放了一套完整脱敏、高品质、符合本项目全部规范（简体中文、文字层级、产品语言、无营销黑话）的历史真实项目交付物示例。

## 示例目录结构

```
examples/
├── README.md               # 本文档
├── ppt/                    # 汇报级幻灯片示例文案
├── prd/                    # 核心产品需求文档 (PRD) 示例
├── sr/                     # 系统需求规格书 (SR) 示例
├── roadmap/                # 时间轴与路线图排期示例
├── research/               # 用户与行业研究报告示例
└── prototype/              # 可运行的交互式线框原型示例 (HTML/CSS)
```

## 学习与参考规则
* **只学风格不抄数据**：本目录中的所有参数、逻辑与表格格式仅作为 AI Agent 进行风格迁移（Style Transfer）的样本。
* **脱敏红线**：严禁在此目录中上传任何包含公司内部系统真实 Token、未公开商业计划与私有 IP 资产的文件。
* **可直接运行原型**：`prototype/` 下的 HTML 原型文件支持直接双击在浏览器中查看焦点交互效果。
