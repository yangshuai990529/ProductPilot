# 示例 - 可编辑交互原型 (Prototype Example)

本目录提供了一个可以完全在本地运行、部署在 GitHub Pages 或 Vercel 上的交互式线框原型。

## 原型文件说明
* **[index.html](file:///Users/eric/Downloads/PPT/Github/ProductPilot/examples/prototype/index.html)**: 包含智能电视主界面（Launcher）及设置菜单树的原型实现，支持键盘 Tab/方向键 模拟遥控器操作。

## 运行方式
1. 直接在文件管理器中**双击**运行 [index.html](file:///Users/eric/Downloads/PPT/Github/ProductPilot/examples/prototype/index.html)。
2. 使用任何现代浏览器（Chrome, Safari, Firefox, Edge）打开即可查看。
3. 部署到服务器或托管平台时，确保所有静态资源路径为**相对路径**。
